## **목표**

사용자들이 여행지 정보를 손쉽게 탐색하고 리뷰를 작성하며, 이를 기반으로 개인 맞춤형 여행 플래너를 생성할 수 있는 플랫폼 개발.

---

## **주요 특징**

1. 여행지 중심의 사용자 리뷰 제공
2. 사용자 선호 데이터를 기반으로 여행지 추천.
3. 개인화된 프로필 및 여행 기록 관리.
